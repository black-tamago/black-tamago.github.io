# 前提提要

Hexo 是一个用 nodejs 开发的博客框架，用于吧 markdown 文件转换成静态网站所需的文件（如 html）。本文章将使用 shoka 作为 hexo 的主题。
这里只探讨在 windows11 环境下的安装和配置。
配置 Hexo 要安装的软件如下

  • nodejs 运行 hexo
  • git 用于下载 shoka（如果没有也可以直接从 github 上下载.zip）

# 生成博客

首先找到个风水宝地（空的文件夹里）用来生成我们的初始文件。对空旷的地方右键并打开命令行或者打开命令行后用 cd 命令移动到风水宝地。
用来生成博客的命令是这个：

跑完后可以运行 npx hexo s 来验证是否生成，让后 Ctrl+C 退出。
生成的文件如下：

.github
node_modules
scaffolds
source
themes
.gitignore
db.json
package-lock.json
package.json
_config.landscape.yml
_config.yml

接下来让我们打开 _config.yml （推荐用类似 vscode 之类的文本编辑器），这个文件是设置一些默认设置的，比如标题，作者名字之类的。
我会列举一些对于 shoka 这个主题比较重要的选项：

title: # 标题
subtitle: # 副标题
author: # 作者

# 关闭代码高亮（因为有一些bug），hexo版本使用命令``` hexo -v ```查看
# hexo版本 <= 7.0
syntax_highlighter:  # empty
# hexo版本 > 7.0
highlight:
  enable: false
prismjs:
  enable: false

theme: # 主题 (shoka)

# 下载 shoka

在风水宝地下打开命令行，让后运行一下其中一个命令即可：

# cd your-blog
git clone https://github.com/amehime/hexo-theme-shoka.git ./themes/shoka
# 后者是我修改的一个版本，修改了一些bug
git clone https://github.com/black-tamago/hexo-theme-shoka.git ./themes/shoka

# shoka 设置

shoka 的设置文件在 风水宝地/theme/shoka/_config.yml ，但是为了方便日后的更新（虽然 2 年没更新了），我们还是复制一份到风水宝地并改名为 _config.shoka.yml 。
